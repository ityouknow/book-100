## 排序规则

- 1、去掉豆瓣评分小于8.5的书籍。
- 2、去掉豆瓣评分人数少于300人的书籍。
- 3、去掉出版时间在2000年之前的书籍。
- 4、评分：评分规则=投票人数+豆瓣评分。
- 5、根据评分的分数排序。
